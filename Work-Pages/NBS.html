NbS Website

The final project in this portfolio currently is another website that was created in a group project. For this group project we were tasked with creating a homepage of a website for a client with a large focus on scrollytelling elements. For this group project I will just focus on elements that I created for the project. My role in this project was the main coder and wireframe creator.
