JavaScript Experiments

JavaScript Experiments was a university project in which I create three JavaScript experiments inside the p5.js environment. The experiments included a clock, mirror using the camera and a painter. The goal of this was to expand my JavaScript knowledge.

The design process for this project was coming up with three original ideas using the three requirements. There is no real documentation that occurred outside the p5.js environment as the project entirely existed inside the p5.js environment.
